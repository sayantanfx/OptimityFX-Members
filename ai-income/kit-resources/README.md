## 🧭 How to import the editable files into Notion
1. Open Notion → click **Import** (bottom-left sidebar).
2. Choose **Markdown & CSV**.
3. Select the `.md` file (becomes an editable page) or `.csv` file (becomes an editable table/database).
4. Edit freely — it's now yours.

## ✏️ How to edit the Word & Excel files
- `.docx` → opens in **Microsoft Word, Google Docs, or Pages**. Replace the `[bracketed]` text.
- `.xlsx` → opens in **Excel, Google Sheets, or Numbers**. Only edit the highlighted input cells; formulas handle the math.
